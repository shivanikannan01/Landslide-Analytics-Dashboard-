# Landslide Analytics Dashboard

## What is a Landslide?

A landslide is the movement of rock, soil, and debris down a slope due to 
gravity, often triggered by natural forces such as heavy rainfall, earthquakes, 
volcanic activity, or erosion. Landslides are among the most destructive 
natural disasters globally, causing thousands of casualties and billions in 
infrastructure damage every year. They occur across all continents and are 
highly unpredictable, making data-driven analysis critical for early warning, 
disaster preparedness, and risk mitigation.

---

## Overview

This project presents an interactive **Landslide Analytics Dashboard** built 
using **Power BI**, analyzing **500 global landslide events** recorded between 
**2020 and 2024** across five major regions — Americas, Europe, Asia, 
Oceania, and Africa.

The dashboard transforms raw disaster data into actionable insights by 
visualizing casualty trends, damage patterns, trigger types, and severity 
distributions — enabling data-driven understanding of landslide risk worldwide.

---

## Objectives

- Analyze the frequency and impact of landslide events across regions and seasons
- Identify dominant trigger types (Earthquake, Rainfall, Volcanic, Erosion)
- Segment events by severity (High, Medium, Low) and measure casualty impact
- Track monthly and yearly trends in event occurrence
- Flag high-impact events for disaster response prioritization

---
## Dataset

| Field | Description |
|---|---|
| event_id | Unique identifier for each landslide event |
| date | Date of occurrence |
| country | Country where the event occurred |
| region | Global region (Americas, Europe, Asia, etc.) |
| severity | Event severity — High, Medium, or Low |
| trigger | Cause — Earthquake, Rainfall, Volcanic, Erosion |
| casualties | Number of lives lost or affected |
| damage_usd_k | Estimated damage in USD (thousands) |
| season | Season of occurrence |
| high_impact_flag | Binary flag for critical events |

> Dataset: Synthetic dataset modeled on real-world landslide patterns

---

##  Key Insights

- **High severity events** account for 33.4% of all landslides but cause disproportionately higher casualties
- **Earthquake-triggered** landslides are the most frequent trigger type (28.4%)
- **Americas and Europe** record the highest event counts across all severity levels
- **Autumn and Winter** seasons show higher landslide frequency, especially earthquake-triggered events
- Countries like **Italy, DR Congo, China, and Switzerland** appear repeatedly in high-impact events
